# AI-CAREER-NAVIGATOR


**Career Navigator AI** is an AI-powered career guidance platform that helps students and professionals bridge the gap between their current skills and their dream job.

Instead of generic advice, Career Navigator AI creates a **personalized roadmap** based on a user's resume, skills, projects, coding profiles, and target role/package. The platform analyzes strengths, identifies skill gaps, and generates a step-by-step action plan to help users achieve their career goals efficiently.

---

## 🎯 Problem Statement

Many aspiring software engineers and data professionals struggle with:

* Not knowing what skills recruiters actually expect
* Following random roadmaps without a clear goal
* Difficulty identifying gaps in their resume
* Lack of personalized career guidance
* Uncertainty about interview preparation priorities

Career Navigator AI solves these challenges by providing a structured and customized career growth plan.

---

## ✨ Features

### 📄 Resume Analysis

* Upload resume in PDF or DOCX format
* Extract technical skills, projects, education, and experience
* Generate a professional profile summary

### 🎯 Goal-Based Career Planning

* Select target role (SDE, Data Analyst, Data Scientist, ML Engineer, etc.)
* Enter desired salary/package range
* Receive a personalized roadmap to reach the target

### 🔍 Skill Gap Analysis

* Compare current skills with industry requirements
* Identify missing technologies and concepts
* Prioritize learning based on importance

### 🛣️ Personalized Learning Roadmap

* Beginner → Intermediate → Advanced progression
* Weekly and monthly milestones
* Recommended learning sequence

### 💻 Coding Profile Evaluation

* Analyze GitHub, LeetCode, CodeChef, HackerRank profiles
* Assess coding consistency and project quality
* Suggest improvement areas

### 📚 Interview Preparation Guidance

* DSA preparation roadmap
* Core CS subjects recommendations
* Behavioral interview tips
* Resume enhancement suggestions

### 🤖 AI Career Coach

* Ask career-related questions
* Get personalized guidance
* Receive actionable recommendations
